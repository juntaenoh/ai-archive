## **1. 문제 인식**

- 이미지 기반 관절 좌표를 그대로 Unity 아바타 본(Bone)에 매핑했을 때,
    
    사람마다 신체 비율이 달라서 팔이나 다리가 비정상적으로 길어지거나 짧아지는 문제가 발생.
    
- 예: 어깨가 넓은 사람은 팔이 이상하게 뻗고, 다리가 긴 사람은 하반신이 비율이 깨짐.

---

## **2. 초기 접근 (캘리브레이션)**

- 사용자가 측정 시작 전에 기준 포즈(캘리브레이션)를 취하도록 설계.
- 이때 상체·하체를 나누어 각각의 기본 비율(어깨–골반 거리, 다리 길이 등)을 계산.
- 이를 통해 “사용자 고유 비율”을 Skeleton에 반영 → 기본적인 비율 불일치 문제 완화.

---

## **3. 중간 시도 (비율 매핑)**

- 단순 캘리브레이션만으로는 동작 중에 관절이 어색해지는 경우가 있고 
기준 포즈를 반드시 받아야 하는 문제점 때문에 사용자 경험이 하락함
    
    **상반신/하반신 구간별 비율 매핑**을 시도.
    
- 예: 상체는 어깨–팔–손목 비율 기준, 하체는 골반–무릎–발목 비율 기준.
- 하지만 동작이 복잡해질수록 오차 누적 문제가 생김.

---

## **4. 최종 해결책 (벡터 기반 리깅)**

- 최종적으로 **관절 간 벡터(Vector) 값**을 활용하는 방식으로 결정.
- 관절 좌표를 단순 위치 값으로 쓰는 대신, **관절 간 상대 벡터**를 계산하여 Skeleton에 매핑.
- 이로써 사용자 체형이 다르더라도, **동작 자체의 방향성과 비율이 유지**되도록 구현.

---

## **5. 결과**

- 다양한 체형(어깨 넓은 사람, 다리 긴 사람 등)에서도 자연스러운 리깅 가능
- 투명도 조절 / 아바타 회전 기능 flutter 내부에서도 동작

    ```csharp
     private Dictionary<string, Vector3> CalculateTo3D(Dictionary<string, Vector2> front, Dictionary<string, Vector2> side)
        {
            var result = new Dictionary<string, Vector3>();
            float scale = 0.001f;
    
            foreach (var kvp in front)
            {
                if (!side.ContainsKey(kvp.Key)) continue;
                Vector2 f = kvp.Value;
                Vector2 s = side[kvp.Key];
                result[kvp.Key] = new Vector3(-f.x, -f.y * 1.2f, -s.x * 0.5f) * scale;
    
            }
    
            CalculateSpine(result);
    
            return result;
        }
    ```
    
    ## 척추 좌표값 추정
    
    ```csharp
     private void CalculateSpine(Dictionary<string, Vector3> points)
        {
            // 필수 포인트 확인
            if (!(points.TryGetValue("leftShoulder", out var ls) &&
                  points.TryGetValue("rightShoulder", out var rs) &&
                  points.TryGetValue("leftHip", out var lh) &&
                  points.TryGetValue("rightHip", out var rh)))
            {
                return;
            }
    
            // 중심점
            var shoulderCenter = (ls + rs) * 0.5f;
            var hipCenter = (lh + rh) * 0.5f;
    
            // 기존 보정 
            //shoulderCenter.z += 0.03f;
    
            // 3D 시작/끝
            Vector3 start = shoulderCenter;
            Vector3 end = hipCenter;
    
            Vector2 shoulderDir2D = new Vector2(rs.x - ls.x, rs.y - ls.y);
            if (shoulderDir2D.sqrMagnitude < 1e-8f) return;
    
            shoulderDir2D.Normalize();
    
            // 수직(왼쪽) 단위벡터: (-y, x)
            Vector2 shoulderPerp2D = new Vector2(-shoulderDir2D.y, shoulderDir2D.x);
    
            // "아래쪽"을 보장하기 위해, 화면 아래(Vector2.down) 쪽과 동일한 방향으로 맞춘다
            if (Vector2.Dot(shoulderPerp2D, Vector2.down) < 0f)
                shoulderPerp2D = -shoulderPerp2D;
    
            Vector2 hipDir2D = new Vector2(rh.x - lh.x, rh.y - lh.y);
            if (hipDir2D.sqrMagnitude < 1e-8f) return;
    
            hipDir2D.Normalize();
    
            Vector2 hipPerp2D = new Vector2(-hipDir2D.y, hipDir2D.x);
    
            // "위쪽"을 보장하기 위해, 화면 위(Vector2.up) 쪽과 동일한 방향으로 맞춘다
            if (Vector2.Dot(hipPerp2D, Vector2.up) < 0f)
                hipPerp2D = -hipPerp2D;
    
            float shoulderWidth = Vector2.Distance(new Vector2(ls.x, ls.y), new Vector2(rs.x, rs.y));
            float hipWidth = Vector2.Distance(new Vector2(lh.x, lh.y), new Vector2(rh.x, rh.y));
    
            float p1OffsetLen = 0.03f; 
            float p2OffsetLen = 0.03f;
    
            // ── p1, p2 계산 
            Vector3 p1 = new Vector3(
                shoulderCenter.x + shoulderPerp2D.x * p1OffsetLen,
                shoulderCenter.y + shoulderPerp2D.y * p1OffsetLen,
                shoulderCenter.z
            );
    
            Vector3 p2 = new Vector3(
                hipCenter.x + hipPerp2D.x * p2OffsetLen,
                hipCenter.y + hipPerp2D.y * p2OffsetLen,
                hipCenter.z
            );
    
            // 결과 키포인트 t 값 
            float upperChestT = 0.30f;
            float chestT = 0.50f;
            float spineT = 0.80f;
    
            points["shoulderCenterOriginal"] = shoulderCenter;
            points["shoulderCenter"] = shoulderCenter + new Vector3(0, 0, 0.02f);
            points["hipCenterOriginal"] = hipCenter;
            points["hipCenter"] = hipCenter + new Vector3(hipPerp2D.x * 0.02f, hipPerp2D.y * 0.02f, 0);
    
            // 곡선 위 포인트 샘플링
            points["UpperChest"] = CubicBezier(start, p1, p2, end, upperChestT);
            points["Chest"] = CubicBezier(start, p1, p2, end, chestT);
            points["Spine"] = CubicBezier(start, p1, p2, end, spineT);
        }
    ```
    
    ## cubic-bezier
    
    ```csharp
       private Vector3 CubicBezier(Vector3 p0, Vector3 p1, Vector3 p2, Vector3 p3, float t)
        {
            t = Mathf.Clamp01(t);
    
            float u = 1f - t;   // (1 - t)
            float uu = u * u;    // (1 - t)^2
            float tt = t * t;    // t^2
    
            //   B0 = (1-t)^3, B1 = 3(1-t)^2 t, B2 = 3(1-t) t^2, B3 = t^3
            return (uu * u) * p0 +
                   (3f * uu * t) * p1 +
                   (3f * u * tt) * p2 +
                   (tt * t) * p3;
        }
    ```

    ```csharp
    private Dictionary<string, Vector3> CalculateDirections(Dictionary<string, Vector3> points)
        {
            var vectors = new Dictionary<string, Vector3>();
            void AddVec(string name, string a, string b)
            {
                if (points.ContainsKey(a) && points.ContainsKey(b))
                    vectors[name] = (points[b] - points[a]).normalized;
            }
    
            AddVec("Hips", "hipCenterOriginal", "hipCenter");
    
            AddVec("UpperChest", "UpperChest", "shoulderCenterOriginal");
            AddVec("Chest", "Chest", "UpperChest");
            AddVec("Spine", "Spine", "Chest");
    
            AddVec("LeftShoulder", "shoulderCenter", "leftShoulder");
            AddVec("RightShoulder", "shoulderCenter", "rightShoulder");
            AddVec("LeftUpperArm", "leftShoulder", "leftElbow");
            AddVec("LeftLowerArm", "leftElbow", "leftWrist");
            AddVec("RightUpperArm", "rightShoulder", "rightElbow");
            AddVec("RightLowerArm", "rightElbow", "rightWrist");
            AddVec("LeftUpperLeg", "leftHip", "leftKnee");
            AddVec("LeftLowerLeg", "leftKnee", "leftAnkle");
            AddVec("RightUpperLeg", "rightHip", "rightKnee");
            AddVec("RightLowerLeg", "rightKnee", "rightAnkle");
    
            return vectors;
        }
    ```
    

## 📌 프로젝트 개요

재직 중 진행한 프로젝트. Vision 랜드마크를 통해 사용자의 신체 움직임을 실시간으로 감지하고, Unity 아바타에 반영하는 시스템 개발.

## ⚙️ 기술 스택

- Unity / C# — 아바타 리깅 및 실시간 렌더링
- Flutter — 연동 앱 UI 개발

## 🔍 핵심 도전 과제

- 비전 랜드마크 정확도 및 노이즈 처리 — 신체 감지 시 떨림/오인식으로 아바타가 불안정하게 움직이는 문제 발생

## 💡 배운 점 / 회고

- 실시간 처리에서 노이즈 필터링의 중요성을 체감 — 정확도와 반응속도 사이의 트레이드오프 고려 필요
- Flutter ↔ Unity 간 데이터 파이프라인 설계 경험
