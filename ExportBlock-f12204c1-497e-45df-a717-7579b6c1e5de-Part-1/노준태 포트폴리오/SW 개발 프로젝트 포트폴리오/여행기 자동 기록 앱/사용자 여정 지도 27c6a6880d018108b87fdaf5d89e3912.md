## 1-2. 주요 페르소나 정보

---

> **주요 페르소나 : 김민지 (27세, 직장인)**
> 
- 사용자 배경 :
    
    사무직 3년 차로 일상에 지쳐 리프레시가 필요함. 감정 표현과 기록을 중요시하며, SNS 공유도 즐김. 최근 권태기를 느끼고 있으며, 자신을 돌보고 싶은 니즈가 커짐.
    
- 사용자 스토리 :
    
    주말에 우연히 혼자 떠난 소도시 여행에서 감정적인 치유를 경험했지만, 그 순간들을 잘 기록하지 못해 아쉬움이 큼.
    
- 사용자 목표 :
    - 여행 중의 감정 변화와 기록을 시각적으로 남기고 싶음
    - 내 감정과 유사한 사람들의 기록도 보고 싶음
    - "여행이 나를 어떻게 바꾸는지" 확인하고자 함

# 2. 사용자 시나리오 (페르소나 기반)

---

## 2-1. 김민지의 강릉 혼자 여행 시나리오

---

### **Day 1: 여행 시작 (오전 10시)**

**상황**: 서울에서 강릉으로 향하는 KTX 안

```
1. 앱 실행 → "여행 기록 시작" 버튼 클릭
2. "강릉역 근처에서 여행이 시작되었어요" 자동 알림
3. 기차에서 바라본 풍경에 감동 → "설렘 가득한 시작"이라고 텍스트 입력
4. AI가 "#설렘, #새로운시작" 태그와 함께 "새로운 여행의 시작, 마음도 덩달아 들뜬다" 문장 생성
5. 첫 번째 카드 자동 생성 완료
```

### **Day 1: 정동진 해변 (오후 2시)**

**상황**: 정동진 해변에 도착해서 30분간 머물며 바다를 바라봄

```
1. 위치 이동 감지 → "정동진해변 근처에 도착했어요" 알림
2. 바다를 보며 느낀 감정 → 음성으로 "진짜 시원하다, 답답했던 마음이 뻥 뚫리는 것 같아"
3. STT 변환 후 AI 분석 → "#시원함, #해방감, #치유" 태그 생성
4. "끝없이 펼쳐진 바다, 답답했던 마음도 함께 날려보낸다" 감정 문장 생성
5. 해변 사진 촬영 → 자동으로 카드에 연결
6. 두 번째 카드 완성
```

### **Day 1: 강릉 중앙시장 (오후 5시)**

**상황**: 중앙시장에서 저녁 식사 및 쇼핑

```
1. 시장 도착 → 자동 위치 감지
2. 북적거리는 분위기에서 → "사람들이 많지만 왠지 정겨워" 입력
3. AI가 "#정겨움, #활기" 태그 생성
4. 순두부찌개와 감자전 사진 촬영 → 카드에 자동 추가
5. 세 번째 카드 완성
```

### **Day 2: 여행 종료 및 여행기 완성 (오후 6시)**

**상황**: 서울로 돌아가는 길, 기차 안에서 여행 기록 종료

```
1. "여행 기록 종료" 버튼 클릭
2. 전체 여행 요약 확인: "강릉 힐링 여행 / 1박 2일 / 총 3개 구간"
3. 여행기 제목을 "혼자만의 시간, 나를 찾아가는 강릉 여행"으로 설정
4. 공개 설정을 "전체 공개"로 선택
5. 완성된 여행기를 인스타그램 스토리로 공유
6. "다음에도 이런 여행 하고 싶다" 만족감 표현
```

## 2-2. 여행기 탐색 및 영감 획득 시나리오

---

### **2주 후: 다음 여행 계획**

**상황**: 김민지가 강릉 여행 이후 다른 지역 여행을 계획하며 앱 사용

```
1. 탐색 탭 접근 → 개인화 피드에서 "#치유, #혼자여행" 관련 여행기 자동 추천
2. "제주도 올레길에서 찾은 나만의 시간" 여행기 발견
3. 여행기 상세 보기 → "나도 이런 느낌의 여행을 하고 싶다" 공감
4. 좋아요 표시 + 북마크 저장
5. 해당 여행기와 유사한 제주 여행 콘텐츠 추가 추천 받음
6. "제주도 계획 세워봐야겠다" 결심
7. 제주도 관련 여행기들 추가 탐색
```
