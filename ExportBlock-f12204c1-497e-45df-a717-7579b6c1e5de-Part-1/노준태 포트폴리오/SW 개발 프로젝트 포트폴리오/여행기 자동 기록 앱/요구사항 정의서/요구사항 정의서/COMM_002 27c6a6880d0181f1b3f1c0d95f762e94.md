# COMM_002

상태: 초안 작성
구분: 커뮤니티
기능 간단 설명: 종료된 단일 여행기에 대한 SNS 공유용 링크 생성 및 복사 버튼 제공
기능명: 여행기 공유
우선 순위: 3순위

| 항목 | 내용 |
| --- | --- |
| **기능 ID** | COMM_002 |
| **기능 명** | 여행기 공유 |
| **기능 설명** | 사용자는 작성한 여행기를 외부와 공유하거나, 다른 사용자의 여행기에 댓글을 작성할 수 있다. 공유 시에는 고유한 링크가 생성되고, SNS 또는 메시지앱 등을 통해 공유 가능하다. 댓글 기능을 통해 감정 공감 또는 여행 정보 교류가 가능하다. |
| **사용자 액션** | 1. 공유 버튼 클릭 → 링크 복사 또는 SNS 선택2. 댓글 입력창에 텍스트 작성 → 등록 클릭3. 댓글 목록 확인 및 좋아요 클릭 |
| **시스템 동작** | 1. 공유 버튼 클릭 시 고유 URL (`/share/{travel_id}`) 생성
2. 선택한 플랫폼(SNS, 카카오톡 등)에 맞춰 링크 전달
3. 댓글 입력 시 사용자 인증 확인 후 DB 저장
4. 댓글 목록은 시간순 또는 인기순으로 정렬 |
| **입력 데이터** | 공유 요청: travel_id댓글 입력: user_id, travel_id, comment_text |
| **출력 데이터** | 공유 링크(URL), 댓글 리스트 (`comment_id`, `작성자`, `내용`, `시간`) |
| **상태 전이** | 없음 |
| **예외 처리** | 비로그인 상태 → 댓글 작성 불가, 로그인 유도공유 실패 시 → “링크를 생성할 수 없습니다” 안내욕설/부적절 표현 → 필터링 및 신고 처리 연동 |
| **비고** | 댓글 수, 좋아요 수를 통해 인기 여행기 정렬 가능공유된 여행기의 공개 범위 설정 기능(전체/비공개/링크만) 고려 필요 |

### 🧠 UX 흐름 예시

```
[사용자] 여행기 상세 보기 → “공유” 버튼 클릭
→ 링크 복사 or SNS 선택 (예: 인스타, 카카오톡)
→ 고유 링크 생성 및 공유 성공 안내

[다른 사용자] 공유된 여행기 진입
→ 아래로 스크롤하면 댓글 섹션
→ “너무 공감돼요!” 댓글 작성 후 등록
→ 댓글 리스트 최신순으로 표시됨
```
