# ARCHET TYPE

### 1. 태그 기반 여행자 아키타입 분류

---

| 아키타입 | 설명 | 대표 태그 (중복 포함) |
| --- | --- | --- |
| 🌿 **고요한 회복러** | 조용하고 한적한 자연 속에서 감정을 정리하고 자신을 회복하려는 성향. 사람보다 ‘공간’과 ‘쉼’에 집중함. | `#고요함`, `#위로`, `#따뜻함`, `#숲속`, `#혼자여행`, `#쉼여행`, `#힐링여행`, `#명상`, `#자연속에서`, `#차분한톤`, `#청명함`, `#이른아침`, `#비오는날`, `#무계획`, `#첫눈`, `#치유` |
| 🎨 **감성 수집가** | 순간의 감정, 분위기, 색감 등을 기록하며 감성적인 콘텐츠 생성에 몰입하는 스타일. 사진·글귀·공간을 감각적으로 포착함. | `#설렘`, `#그리움`, `#행복`, `#노을`, `#햇살`, `#골목길`, `#카페탐방`, `#필름감성`, `#레트로`, `#로컬감성`, `#사진중심`, `#텍스트중심`, `#잔잔함`, `#혼자여행`, `#감성기록`, `#1일1기록` |
| 🧭 **탐험형 활동러** | 즉흥적으로 장소를 이동하며 활동과 발견에 몰두하는 타입. 새로운 장소나 미지의 장소 탐색에 열려 있음. | `#신남`, `#트레킹`, `#즉흥여행`, `#시장구경`, `#산책`, `#전망대`, `#도보여행`, `#한옥마을`, `#성지순례`, `#여름밤`, `#계획없음`, `#핵심스팟`, `#새로운장소`, `#에너지` |
| 🧳 **가족 중심 케어러** | 가족 구성원(아이, 부모)과 함께 안정적이고 편안한 여행을 선호. 휴식과 함께 주변을 챙기는 역할을 자주 수행. | `#가족여행`, `#아이들과함께`, `#부모님과함께`, `#공원`, `#편안한코스`, `#전통시장`, `#사진여행`, `#계획여행`, `#쉼여행`, `#느긋함`, `#맛집탐방`, `#테마파크`, `#기획여행` |
| 📸 **풍경 마스터** | 자연 풍경, 빛, 계절감 등을 중시. 여행지의 시각적 아름다움과 순간을 정적인 방식으로 담아내는 경향. | `#자연치유`, `#전망대`, `#청명함`, `#노을`, `#파도소리`, `#달빛`, `#해변`, `#겨울바다`, `#가을단풍`, `#봄꽃`, `#하늘`, `#풍경중심`, `#사진중심`, `#따뜻한톤`, `#첫눈`, `#비오는날` |
| 💬 **소셜 공감러** | 사람들과 감정을 나누고, 공감받고 싶은 욕구가 큼. 공유·소통 중심으로 콘텐츠를 소비하고 생성함. | `#공유일기`, `#좋아요많음`, `#댓글많음`, `#친구들과`, `#커플여행`, `#가족여행`, `#감성공유`, `#여행공감`, `#포토기록`, `#소통`, `#같이걷기`, `#여행일기`, `#1일1기록` |
| 🎯 **목적 집중형** | 명확한 목표(맛집, 포토스팟, 특정 장소 방문 등)를 설정하고, 효율적 이동과 성취를 중시함. 계획적 성향이 강함. | `#사진여행`, `#맛집투어`, `#성지순례`, `#테마파크`, `#기획여행`, `#계획여행`, `#맛집중심`, `#핵심스팟`, `#전통시장`, `#산책루트`, `#문화유산`, `#자전거`, `#핫플투어` |
| 🧘 **느린여행자** | ‘일정 없는 자유’ 그 자체를 즐기며, 조용하고 의미 있는 시간을 혼자 또는 한적한 공간에서 보냄. | `#쉼여행`, `#무계획`, `#자연속에서`, `#산책`, `#혼자여행`, `#이른아침`, `#한낮의산책`, `#느긋함`, `#비오는날`, `#조용한장소`, `#따뜻한톤`, `#고요함`, `#차분한톤` |

### 4. 향후 확장 가능

---

| 확장 방향 | 설명 |
| --- | --- |
| ✍ 성향 테스트 기반 진입 | "나의 여행 성향 테스트" (MBTI-lite 스타일) |
| 📦 추천 큐레이션 강화 | 아키타입별 추천 장소/루트/사진 템플릿 |
| 🧠 자동 분류 | 사용자의 감정 벡터 + 장소 로그 → 자동 군집화 |
| 🎁 마케팅 활용 | “감성 수집가를 위한 3일 속초 코스”, “회복러를 위한 혼자 제주 추천” |
