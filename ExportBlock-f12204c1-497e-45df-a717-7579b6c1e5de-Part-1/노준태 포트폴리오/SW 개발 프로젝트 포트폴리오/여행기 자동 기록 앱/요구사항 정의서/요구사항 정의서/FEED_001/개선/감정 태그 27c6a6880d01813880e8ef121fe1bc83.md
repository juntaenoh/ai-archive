# 감정 태그

---

장점

- 공감 포인트 제공 / 스토리 전달력 강해짐
- 감정 기반 추천 / 감정별로 모아볼 수 있는 콘텐츠 제작 가능
- 심리적 치유, 소통 요소
    - 여행 플랫폼 트렌드 중 하나는 웰니스/마음 돌봄 여행인데, 감정 태그는 이 부분에 잘 맞을 수도

단점

- 부정적 감정 노출 부담
    - `#짜증` `#우울` 같은 태그는 플랫폼 전체 분위기를 다소 무겁게 만들 수 있음
- 표현의 다양성 부족
    - 감정은 미묘한데, 태그만으로 단편적으로 표현하기에 어려움

대안

- **감정을 직접 태그화하되 톤 조절**
    - 긍정적/중립적/부정적 감정을 모두 포함하되, 노골적인 표현 대신 조금 더 여행 톤에 맞는 단어를 사용
    - 예
        - `#짜증` → `#불편함`
        - `#우울` → `#조용한마음` `#지친하루`
        - `#기쁨` → `#순간의행복` `#즐거움`
