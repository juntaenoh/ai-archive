# COMM_001_1

상태: 초안 작성
구분: 커뮤니티
기능 간단 설명: 대표 이미지, 감정 태그, 제목으로 구성된 콘텐츠 카드 UI 제공
기능명: 카드 구성
상위 항목: COMM_001 (COMM_001%2027c6a6880d0181db96fec4a7a7f76404.md)

| 항목 | 내용 |
| --- | --- |
| **기능 ID** | COMM_001_1 |
| **기능 명** | 기록 카드 구성 (기능) |
| **기능 설명** | 사용자의 여행 로그 및 정리 정보를 기반으로 콘텐츠 카드 단위를 자동 구성한다. 대표 이미지, 감정 문장, 감정 태그, 장소 정보, 시간대 등 핵심 데이터 요소들을 시각적 카드 UI로 가공해 여행 피드와 추천 목록에 출력된다. 구성 요소는 사용자의 성향(아키타입) 또는 기록의 특성에 따라 자동으로 가중치를 달리하여 표현된다. |

### ⚙️ 주요 기능 요소

| 기능 요소 | 설명 | 구현 방식 예시 |
| --- | --- | --- |
| **대표 이미지 자동 선택** | 여행기 내 다수 이미지 중 대표 이미지 1장 선택 | 최근 업로드 순 / 가장 공감 많은 사진 / AI 이미지 품질 평가 |
| **감정 문장 선택 우선순위** | 감성 문장 > 사용자 입력 문장 > 태그 조합형 문장 | 조건 분기 로직 + fallback 처리 |
| **태그 정렬 및 필터링** | 감정, 활동, 시간대 태그 중 2~3개만 노출 | 태그 우선순위 부여 + 짧은 키워드 우선 |
| **장소 정보 표기** | 장소명 + 시간 (예: “속초해변 · 오전 10:00”) | 여행 로그의 위치 + 시간 기반 포맷 구성 |
| **반응 수 표기** | 좋아요 수, 댓글 수를 간단히 카드 하단에 노출 | 메타데이터 포함된 summary object 활용 |
| **카드 유형 템플릿 분기** | 사용자 아키타입별 레이아웃 다르게 적용 | 감성 수집가 → 사진 강조형, 목적 집중형 → 제목 강조형 |
| **빈 데이터 대응** | 문장 없음, 이미지 없음, 태그 없음 시 대체 콘텐츠 표시 | default 문장 / 배경 테마 / ‘#여름여행’ 등 |
| **탭별 필터 대응** | ‘감정별’, ‘장소별’, ‘내 취향 기반’ 피드에 적합한 카드 타입으로 구성 | 요청 URI context 기반 카드 렌더링 방식 변경 |

### 데이터 구조 예시 (카드 JSON)

```json
{
  "card_id": "card_001",
  "title": "속초해변",
  "sentence": "잔잔한 바닷바람이 마음까지 시원해졌다",
  "tags": ["#노을", "#고요함", "#산책"],
  "image_url": "https://cdn.example.com/travel123/cover.jpg",
  "time": "오전 10:00 ~ 11:30",
  "location": "속초해변",
  "likes": 21,
  "comments": 3,
  "card_type": "emotion_image_card"
}
```

### 카드 구성요소

| 구성 요소 | 설명 | 예시 |
| --- | --- | --- |
| 📷 대표 이미지 | 해당 여행 구간의 대표 사진 | 썸네일로 배치 |
| 🧭 장소명 | 근사 관광지명 or 사용자가 지정한 위치명 | “속초해변”, “경주 중앙시장” |
| 💬 감성 문장 | AI가 생성한 감정 요약 문장 | “잔잔한 바닷바람, 마음까지 시원해졌다” |
| 💬 입력 문장  | 사용자가 입력한 문장  | “아이들과 산책하기 너무 좋았어요”  |
| 🏷 감정 태그 | 감정 기반 해시태그 | #고요함 #햇살 #자유 |
| 🕒 시간 | 구간 시간 or 여행 날짜 | “오전 10:00 ~ 11:30” |
| 👤 작성자 정보 | 닉네임 + 프로필 이미지 | “감성수집가 🌿” |
| 💗 반응 수 | 좋아요/공감 수 | 💗 21, 💬 3 |
| 🔗 상세 보기 | 카드 클릭 → 상세 페이지로 이동 |  |
