# RECD_005

상태: 초안 작성
구분: 여행기 기록
기능 간단 설명: 여행 기록 중, 사용자가 원하는 시점에 텍스트로 감정을 작성하고 기록
기능명: 감정 문장 기록
우선 순위: 1순위

| 항목 | 내용 |
| --- | --- |
| **기능 ID** | RECD_005 |
| **기능 명** | 감정 문장 기록 |
| **기능 설명** | 여행 중 감정 입력 또는 여행 정리 시점에, 날씨/시간대/사용자 입력 감정 키워드를 종합적으로 분석하여 감성적인 문장을 자동 생성하고 해당 여행기 카드에 삽입한다. 이를 통해 여행 콘텐츠의 몰입도를 높이고, 감정 기반 공유 콘텐츠로서의 가치도 향상시킨다. |
| **사용자 액션** | 1. 감정 한 마디 입력 (예: “시원하다”)2. 사진 업로드3. 여행 종료 후 자동 카드 정리 |
| **시스템 동작** | 1. 감정 키워드 + 시간대(예: 오전 10시) + 날씨 정보(예: 맑음, 23도)를 기반으로 감성 문장 생성 요청
2. AI 문장 생성기(GPT 등)를 통해 한 문장 요약 문장 반환
3. 해당 일정 구간(card)에 삽입
4. 수동 감정 문장이 있을 경우 덮어쓰기 여부 확인 |
| **입력 데이터** | 감정 키워드, 시간대, 기온 및 날씨 상태, 장소명 (선택) |
| **출력 데이터** | 감성 문장 1줄 (`sentence_text`) → ex: “잔잔한 햇살 아래, 바닷바람이 마음까지 맑게 씻겨가는 느낌” |
| **상태 전이** | 없음 |
| **예외 처리** | 감정/날씨 데이터 미존재 시 → 기본 템플릿 문장으로 대체AI 응답 실패 → “감정 문장을 불러오지 못했어요” 메시지 표시 + 재생성 버튼 |
| **비고** | AI 문장 생성 결과는 감정 태그와 일관성을 유지해야 함장소명/풍경 기반 스타일 프롬프트 확장 가능 |
