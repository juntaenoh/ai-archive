# COMM_001_2

상태: 초안 작성
구분: 커뮤니티
기능 간단 설명: 전체 이동 경로와 주요 장소를 지도 위에 시각적으로 표시
기능명: 단일 여행기 상세 조회 (지도 기반 조회)
우선 순위: 2순위
상위 항목: COMM_001 (COMM_001%2027c6a6880d0181db96fec4a7a7f76404.md)

| 항목 | 내용 |
| --- | --- |
| **기능 ID** | COMM_001_2 |
| **기능 명** | 지도 기반 보기 |
| **기능 설명** | 사용자의 여행 전체 경로와 각 일정 구간의 주요 장소들을 지도 위에 시각적으로 표시한다. 마커와 이동 경로(폴리라인)를 기반으로 시공간의 흐름을 직관적으로 보여주며, 감정 카드 또는 사진과 연동된 지도 UI를 제공하여 장소 중심의 회고와 탐색을 돕는다. |
| **사용자 액션** | 1. 상세 보기 또는 마이페이지에서 “지도 보기” 선택2. 날짜/구간 필터 선택3. 지도 마커 클릭 시 카드 요약 보기 |
| **시스템 동작** | 1. travel_id 기반의 일정 구간(trip_section) 데이터 로드
2. 각 구간의 대표 장소에 대해 장소 좌표(place_coord) 확인
3. 장소 순서대로 폴리라인(이동 경로) 생성
4. 각 장소에 마커 표시 + 감정 요약 또는 대표 사진 썸네일 포함
5. 마커 클릭 시 해당 구간 카드 or 상세 내용 팝업 |
| **입력 데이터** | travel_id, trip_section, place_name, place_coord (위경도), 감정 요약, 사진 썸네일 |
| **출력 데이터** | 지도 UI, 마커 정보, 경로 선(Polyline), 팝업 카드 내용 |
| **상태 전이** | 없음 |
| **예외 처리** | 좌표 없는 장소 → 지도에 표시하지 않음. 
좌표 중복 시 마커 겹침 방지.
배치지도가 로드되지 않을 경우 에러 문구 및 새로고침 안내 |
| **비고** | 지도 UI는 Kakao Map, Leaflet, 또는 Mapbox 등 외부 라이브러리 사용 예정날짜 필터링, 감정 태그 강조 등과 연계하여 지도 내 표현 강화 가능 |

## 🧠 UX 예시 흐름

---

```
[사용자] 마이페이지 → “여행기 보기” → “지도 보기” 선택
→ 지도에 마커 5개 + 경로 선 표시
→ 마커 클릭 → “속초해변 – 시원한 아침” 카드 요약 팝업
→ 날짜 필터로 1일차만 보기
```

## ✅ 지도 내 주요 표현 요소

---

| 요소 | 설명 |
| --- | --- |
| 📍 마커 | 각 구간의 대표 장소 좌표 기준 (장소명, 감정 이모지 등 포함 가능) |
| ➖ 경로 선 | 시간 순서대로 연결된 장소 간 이동 경로 |
| 🖼 팝업 | 마커 클릭 시 카드 요약 정보 (감정, 사진, 시간 등) |
| 📅 날짜 필터 | 다일차 보기 필터 가능 (예: 1일차, 2일차 분리) |
| 🎨 감정 강조 | 특정 감정 태그 선택 시 해당 마커 강조 표시 (#고요함 → 파란색 강조 등) |
