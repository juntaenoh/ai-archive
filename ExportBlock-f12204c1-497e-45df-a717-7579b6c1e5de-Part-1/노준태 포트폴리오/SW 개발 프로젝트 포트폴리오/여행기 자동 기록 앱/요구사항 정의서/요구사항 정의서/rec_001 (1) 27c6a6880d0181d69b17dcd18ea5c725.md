사용자는 자신과 감성적으로 유사한 여행기, 장소, 감정을 가진 다른 사람들의 기록을 피드에서 탐색할 수 있으며, AI 기반 추천 알고리즘이 주기적으로 피드를 갱신한다. |

| **사용자 액션** | 1. 홈 화면 진입
2. 피드 스크롤 탐색
3. 필터(감정, 지역, 태그) 선택 |
| **시스템 동작** | 1. 사용자 ID로 관련 기록 로드 (감정, 태그, 장소)
2. 유사 감정/태그/장소 기반 피드 후보군 구성
3. 추천 알고리즘 기반으로 점수화 → 정렬
4. 피드 카드 형태로 사용자에게 노출 |
| **입력 데이터** | user_id, user_emotion_vector, 관심 태그, 지역 선호도 |
| **출력 데이터** | 피드 카드 리스트 (`travel_card_id`, `대표 이미지`, `감정 태그`, `작성자`, `장소`, `시간`) |
| **상태 전이** | 없음 |

`feed_001`: 여행기 카드 시각화 기반으로 구성

| **비고** | 추천 점수는 감정 유사도 + 장소 유사도 + 공감 수 가중치로 계산탐색 결과를 기반으로 사용자 취향 벡터를 업데이트함 |

```jsx
**[사용자 여행기 기록]**

→ 감정 유사 + 장소 유사 콘텐츠 상위 정렬

```

# *가중치 선정 기준*

---

***사용자에게 피드가 노출될 때, 반영되면 좋을 것 같다 생각***

*1. 지역* 

*⇒ 사용자가 관광하고 싶은 지역에 대해 알아볼 수 있는 구조여야 함*

*2. 태그*

*⇒ 사용자가 어떤 분위기의 관광지를 좋아하는지 알아야 함* 

***가중치 계산 방법을 생각해 봤을 때*** 

*방법 1)* 

*지역, 태그 항목 모두에 가중치를 주고, 일정 시간을 주기로 점수를 계산한 후 우선순위가 높은 것 부터 피드에 노출*

*⇒ 우선 순위의 상위 N개에 따라 피드 노출 비율(%)로 정해서, 피드란에 띄운다.* 

*방법 2)*

*지역, 태그를 1차적으로 필터링할 수 있는 구조 + 방법 1*  

### *가중치 반영 항목*

| 입력 항목 | 설명 |
| --- | --- |
| `user_emotion_vector` | 사용자의 감정 태그 누적 벡터 (예: #설렘 0.8, #고요함 0.6 등) |
| `user_place_vector` | 사용자가 자주 방문하거나 선호하는 지역 |
| `~~user_tag_vector~~` | ~~사용자가 자주 사용한 키워드, 해시태그~~ |
| `global_feed_pool` | 전체 여행기 카드 후보군 |
| `liked_feed_ids` | 사용자가 좋아요 또는 댓글을 남긴 콘텐츠 |

*+ 사용자 아키타입화를 통한 여행 스타일 분류 및 피드 가중치 반영이 들어가도 괜찮지 않을까* 

*⇒ 여행 스타일 분류는 MyPage 통계 + 서비스 풍에 맞도록 꾸미는 목적*

*⇒ 가중치 반영은 비슷한 여행을 선호하는 또 다른 사용자의 여정을 추천하기 위함* 

- *추천 점수 계산 로직 예시*

```python

*score = (emotion_similarity * 0.5) +
        (place_similarity * 0.2) +
        (tag_overlap_score * 0.2) +
        (social_engagement_weight * 0.1)*
```

- ***emotion_similarity**: cosine similarity 또는 tag overlap로 측정*
- ***place_similarity**: 여행 지역이 얼마나 유사한지 (동일 시군구 가중치 ↑)*
- ***tag_overlap_score**: 관심 키워드/해시태그 일치 비율*
- ***social_engagement_weight**: 좋아요 수, 댓글 수 반영 (보정용)*

- *유저 가중치 벡터 예시*

```

*user_emotion_vector = {설렘: 0.9, 고요함: 0.5}
후보 카드 A 감정 = {설렘: 0.8, 자유: 0.3}
→ 감정 유사도 높음 → 상위 노출*
```

### *카드 구성요소*

| 구성 요소 | 설명 | 예시 |
| --- | --- | --- |
| 📷 대표 이미지 | 해당 여행 구간의 대표 사진 | 썸네일로 배치 |
| 🧭 장소명 | 근사 관광지명 or 사용자가 지정한 위치명 | “속초해변”, “경주 중앙시장” |
| 💬 감성 문장 | AI가 생성한 감정 요약 문장 | “잔잔한 바닷바람, 마음까지 시원해졌다” |
| 💬 입력 문장  | 사용자가 입력한 문장  | “아이들과 산책하기 너무 좋았어요”  |
| 🏷 감정 태그 | 감정 기반 해시태그 | #고요함 #햇살 #자유 |
| 🕒 시간 | 구간 시간 or 여행 날짜 | “오전 10:00 ~ 11:30” |
| 👤 작성자 정보 | 닉네임 + 프로필 이미지 | “감성수집가 🌿” |
| 💗 반응 수 | 좋아요/공감 수 | 💗 21, 💬 3 |
| 🔗 상세 보기 | 카드 클릭 → 상세 페이지로 이동 |  |
