## 개요

이 사이트 자체를 만드는 과정을 첫 번째 아카이브 항목으로 기록합니다.
Claude Code를 통해 기획부터 배포까지 전 과정을 진행했습니다.

## 기획

어떤 사이트를 만들지 Claude와 대화하면서 요구사항을 구체화했습니다.

**핵심 결정 사항:**
- 콘텐츠 관리: MDX 파일 기반 (DB 없이 Git으로 관리)
- 배포: Cloudflare Pages (무료 티어)
- 목적: AI 활용 방식 자체를 보여주는 것

처음엔 CMS 연동도 고려했지만, 개발자가 직접 작성하는 아카이브 특성상 MDX + Git이 더 자연스럽다고 판단했습니다.

## 개발

**기술 스택 선정:**

Next.js 16 (App Router) + TypeScript + Tailwind CSS v4를 선택했습니다.
Cloudflare Pages에 배포하기 위해 `@cloudflare/next-on-pages` 설정도 진행했습니다.

**MDX 파이프라인:**

```
content/works/*.mdx
  → gray-matter (frontmatter 파싱)
  → next-mdx-remote/rsc (MDX 렌더링)
  → prose 스타일링 (Tailwind Typography)
```

**주요 AI 활용 포인트:**

1. **컴포넌트 설계**: AI 도구별 색상 뱃지 시스템을 Claude에게 제안 요청
2. **Tailwind v4 대응**: 기존 v3 문법과 달라진 부분을 문서 기반으로 파악
3. **타입 정의**: `WorkMeta` 타입 구조를 Claude와 대화하며 결정

## 배포

Cloudflare Pages와 GitHub 레포를 연결해 자동 배포 파이프라인을 구성했습니다.
`main` 브랜치 푸시 → 자동 빌드 → 배포 완료.
